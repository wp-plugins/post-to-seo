Helps generating the main SEO components Keywords, 160 char description and tags. 

== Description ==

This tools aims to cut down the time of writing the essential SEO components: keywords, description and suggested tags for posts. The plugin parses the text in post editor and displays the suggested keywords, description and tags. 
The process is done via an ajax request so you don't have to reload or leave the page.

Furthermore, you can also optimise this tool by providing Permanent Keywords (that should appear in every post) and Ignored Keywords (that should not be displayed at all) in the Settings section of the plugin.

== Installation ==

1. Extract and upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in the WordPress admin panel
3. SEO generation panel will appear on Post Editor page in the admin area.
4. Go to `Settings->PostToSEO` to add permenant/ignored keywords.
